## Quick Start

Navigate to the `templates/orchestration/` directory to access:
- `redlight.yml` - Critical issue template
- `yellowlight.yml` - Warning/caution template
- `greenlight.yml` - Success/ready template
- `orchestrator.yml` - Orchestration configuration
- `README.md` - Complete documentation

## Usage

1. Assess your current activity state
2. Select the appropriate template (Red/Yellow/Green)
3. Follow the defined actions
4. Monitor transitions and outcomes

For more information, see the [orchestration documentation](templates/orchestration/README.md).
